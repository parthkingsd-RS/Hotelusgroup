# HOTEL US GROUP Website

**HOTEL US GROUP** is a modern hotel located on Delhi-Ajmer Highway, Jaipur.  
This is a fully responsive, dark/night-vibe website with glowing neon effects designed to showcase the hotel, rooms, gallery, and contact information.

## Features
- Dark/night theme with glowing headers and buttons  
- Responsive design for mobile and desktop  
- Rooms and gallery sections with hover animations  
- Contact form for customer inquiries  
- Ready for free hosting on GitHub Pages or Netlify
